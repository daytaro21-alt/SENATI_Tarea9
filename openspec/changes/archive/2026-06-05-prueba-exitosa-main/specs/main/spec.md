## ADDED Requirements

### Requirement: Confirmación de ejecución exitosa

El script `main.js` SHALL imprimir 'Prueba exitosa' en la consola estándar cuando sea ejecutado con NodeJS.

#### Scenario: Ejecución de main.js
- **WHEN** se ejecuta el comando `node main.js` en la consola.
- **THEN** la salida de consola contiene exactamente 'Prueba exitosa'.
