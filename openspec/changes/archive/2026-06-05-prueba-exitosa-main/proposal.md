## Why

Agregar un punto de entrada principal básico (`main.js`) para validar la ejecución exitosa del entorno del proyecto.

## What Changes

- Crear el archivo `main.js` en la raíz del proyecto.
- Configurar el script para que imprima 'Prueba exitosa'.

## Capabilities

### New Capabilities
- `main-execution`: Script principal de NodeJS que confirma la ejecución exitosa del entorno.

### Modified Capabilities
Ninguna.

## Impact

Se agregará un nuevo archivo `main.js` en el directorio raíz.
