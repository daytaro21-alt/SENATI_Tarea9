## Why

Presentar un script simple de NodeJS que imprima un saludo para la clase en la consola.

## What Changes

- Crear el archivo `saludo.js` en la raíz del proyecto.
- Configurar el script para que imprima 'Hola Clase'.

## Capabilities

### New Capabilities
- `saludo`: Script simple de NodeJS para imprimir 'Hola Clase'.

### Modified Capabilities
Ninguna.

## Impact

Se agregará un nuevo archivo `saludo.js` en el directorio raíz.
